# 🔷 RGX | Maplestory Level-Up! ## ![](https://github.com/donniedice/SimpleQuestPlates/blob/main/media/kiwi.gif?raw=true) RGX Mods - RealmGX Community Project *** ## 🎯 Overview **RGX | Maplestory Level-Up!** brings a MapleStory-inspired level-up jingle to World of Warcraft, replacing the default sound with a bright nostalgic level-up moment. ![RealmGX Kiwi](https://github.com/donniedice/SimpleQuestPlates/blob/main/media/kiwi.gif?raw=true) **The Kiwi Says:** "Maple ding! Bwwiiiee." *** ## ✨ Features 🎵 MapleStory Sound - A MapleStory-style level-up jingle plays every time you gain a level. 🎚️ Quality Options - Switch between High, Medium, and Low quality variants. 🔇 Smart Muting - Auto-mutes WoW's default level-up sound so you only hear MSLU. 💬 Login Message - A branded MSLU login message appears on login and can be toggled on or off. 💾 Persistent Settings - Settings are saved between sessions. ⚡ Lightweight - Minimal footprint — no embedded libraries required. 🎮 Multi-Version - Includes support for Retail, Wrath, Burning Crusade, and Classic Era. **Note:** All quality variants are included — no additional downloads required. *** ## 📋 Command Reference The addon can be managed using the following slash commands:
/mslu help # Show all available commands
/mslu test # Play test sound
/mslu enable # Enable the addon
/mslu disable # Disable the addon
/mslu welcome # Toggle login message
/mslu high # Use high quality sound
/mslu med # Use medium quality sound (default)
/mslu low # Use low quality sound
