# 🔷 RGX | Maplestory Level-U! ## 🎯 Overview Maplestory Level Up! brings a MapleStory-inspired level-up sound to World of Warcraft with lightweight settings, slash commands, and multi-version support. ## ✨ Features 🎵 MapleStory Sound - A MapleStory-style level-up jingle plays every time you gain a level. 💬 Login Message - A branded MSLU login message appears on login and can be toggled on or off. 📋 Slash Commands - Use /mslu to test, enable, disable, and change sound quality.
